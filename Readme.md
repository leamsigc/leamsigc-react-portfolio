# LEAMSIGC portfolio web 

# How to run locally 

* git clone https://github.com/leamsigc/leamsigc-react-portfolio.git
* cd leamsigc-react-portfolio

* npm i 

* npm run start 
	This comand will open your default broser and run the app in port http://localhost:1234
